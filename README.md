# Neural Networks
Problem solutions from the Neural Networks course at University of Wrocław.

## Assignment 1
Neural network from scratch, backward pass, gradient descent, ReLU, sigmoid

## Assignment 2
Matrix multiplication on CPU vs GPU, SGD on MNIST, momentum, learning rate schedule, weight decay, hyperparameter tuning, convolutional and max-pooling layers, data augmentation methods, dropout, batch normalization, norm constraints, pruning, gradient clipping

## Assignment 3
Convolutional layers, pooling layers, activation layers and fully connected layers from scratch, testing train set sizes, baseline AlexNet model implementation, input normalization, hyperparameter tuning, SGD and Adam from scratch, batch normalization and dropout from scratch, data augmentation, residual connections, inception modules

## Assignment 4
Object detection using YOLO (model usage, NMS, thresholding, IoU, mAP, Soft-NMS), segmentation using U-Net (U-Net implementation, binary cross entropy, pixel accuracy, IoU, Dice coefficient, ways to deal with varied input sizes, ablation study on residual connections in U-Net)

## Assignment 5
Word2vec from scratch, word embedding using gensim, t-SNE, FastText, Procrustes analysis, PCA, language model from scratch using Simple RNN/LSTM/GRU
